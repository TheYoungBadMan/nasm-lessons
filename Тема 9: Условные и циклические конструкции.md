### **1.1. `if (a == b)`**  

#### **C-–∫–æ–¥:**  
```c

    c = 1;
} else {
    c = 0;

```

### **1.3. `switch-case`**  

#### **C-–∫–æ–¥:**  

### **2.3. `for (i = 0; i < N; i++)`**

üìå `CMP` + `JGE` ‚Üí –≤—ã—Ö–æ–¥ –ø—Ä–∏ `i >= N`.
